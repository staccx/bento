# Cosine interpolate

<Subtitle>
  This document describes the use of the cosineInterpolate function
</Subtitle>

## Requirements

`@staccx/bento >= 14.0.0`

## Usage

```js
import { cosineInterpolate } from "@staccx/bento"
const value = cosineInterpolate(0, 10, 0.5)
```
