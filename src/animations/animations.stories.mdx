# Animations
This story contains several animations that can be used as a global variable in our styled components.

`Protip:` Remember to set the initial placing of the component in addition to the values from the imported keyframe animation! Sometimes when the code is used in production, the animation can "glitch" and make the overall impression bad.

Each animation in this story will include an import-statement.

## AppearIn
This is the AppearIn keyframe with a div.

### Import
All Animations are imported via animations. to use a spesific animation you would
use ```animations.animationYouWant```

```javascript
import { animations } from "@staccx/bento"
```
usage
```javascript
animations.AppearIn
```

  <Story name="AppearIn">
   <AppearComp>AppearIn</AppearComp>
  </Story>

```css
    animation: ${Bounce} 2s ease-out forwards;
    transform: translateX(-100px);
```
