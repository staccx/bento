# useBrRegSearch

### Import
All Hooks are imported via hooks. to use a spesific hook you would
use ```hooks.hookYouWant```

```javascript
import { hooks } from "@staccx/bento"
```
usage
```javascript
hooks.useBrRegSearch
```


## useBrRegSearch setup component

```javascript
import {hooks, List} from "@stacc/bento"
import ReactJson from "react-json-view"
import React from "react"

export const Component = ({ term }) => {
  const { isLoading, results } = hooks.useBrRegSearch(term)

  return (
    <div>
      <List>
        {isLoading
          ? "Loading..."
          : results.map(item => (
              <ReactJson
                key={item.organisasjonsnummer}
                src={item}
                name={`${item}`}
                theme="monokai"
                collapsed
              />
            ))}
      </List>
    </div>
  )
}
```
