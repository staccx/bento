# CreditCardInput
This components controlls user input so that they can only enter numbers with the format of a creditcard number


```javascript
import {CreditCardInput} from "@staccx/bento"
```



## Standard CreditCardInput
The inputfield only allowed numbers and takes number in a 4-4-4-4 format with spacing between and max 16 carrecters
