# Format Date Distance

<Subtitle>
  This document describes the use of the `formatDateDistance` function
</Subtitle>

## Requirements

`@staccx/bento >= 14.0.0`

## Usage

```js
import { formatDateDistance } from "@staccx/bento"
const date = new Date(1525910400000)
const old = new Date(1524614400000)

const formatted = formatDateDistance(date, old)
```
