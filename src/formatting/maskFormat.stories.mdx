# maskFormat
mask format lets you format a string to take the shape you want. The function also allows for prefix and
custom delimiter

### Import
All exports under formatting is imported via formatting. to use a spesific formatter you would
use ```formatting.FormatterYouWant```


```javascript
import { formatting } from "@staccx/bento"
```
usage
```javascript
formatting.maskFormat
```

## Standard maskFormat
maskFormat with the standard mask here formatting a string

<Preview>
  <Story name="Standard">
   <p>{maskFormat("92284082")}</p>
  </Story>
</Preview>

## Passing a mask
Passing a mask to maskFormat will allow you to format the string as you want

<Preview>
  <Story name="Passing mask">
   <p>{maskFormat("92284082", "XXXX XXXX")}</p>
  </Story>
</Preview>


## Passing a mask and prefix
maskFormat also allows you to add prefix to the string like a phoneNumber as you see here

<Preview>
  <Story name="Passing mask and prefix">
   <p>{maskFormat("92284082", "tlf: XXX XX XXX")}</p>
  </Story>
</Preview>

## Custom delimiter
You can also pass a custom delimiter through the mask, replacing the standard space with whatever you like

<Preview>
  <Story name="Custom delimiter">
   <p>{maskFormat("92284082", "XXX-XX-XXX")}</p>
  </Story>
</Preview>
