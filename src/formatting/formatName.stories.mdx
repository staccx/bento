# formatName
Simple name formatter, takes a string and formats it lastname, firstname

### Import
All exports under formatting is imported via formatting. to use a spesific formatter you would
use ```formatting.FormatterYouWant```


```javascript
import { formatting } from "@staccx/bento"
```
usage
```javascript
formatting.formatName
```

## Standard formatName
Here formatName takes the name Jane doe and formats it.
The result will be:

<Preview>
  <Story name="Standard">
   <p>{formatName("Jane doe")}</p>
  </Story>
</Preview>


## Uppercase as input
FormatName will also work from uppercase
input: JANE DOE.
Result:

<Preview>
  <Story name="Uppercase">
   <p>{formatName("JANE DOE")}</p>
  </Story>
</Preview>


## Custom separator
Format name also allows for custom separators

<Preview>
  <Story name="Custom separator">
   <p>{formatName("Jane doe", { separator: "-"})}</p>
  </Story>
</Preview>
