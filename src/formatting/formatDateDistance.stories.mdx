# formatDateDistance
FormatDateDistance lets you pass in two date strings and returns the difference between the two dates


### Import
All exports under formatting is imported via formatting. to use a spesific formatter you would
use ```formatting.FormatterYouWant```


```javascript
import { formatting } from "@staccx/bento"
```
usage
```javascript
formatting.formatDateDistance
```


## Standard formatDateDistance
formatDateDistance takes two dates, and returns the distance between them, in the form of
how long it is from date1 untill date 2.

```javascript
    export const date1 = "2020-02-12T00:00:00"
    export const date2 = "2020-03-19T00:00:00"
    export const distance = formatDateDistance(date1, date2)
```



Will return this:

export const date1 = "2020-02-12T00:00:00"
export const date2 = "2020-03-19T00:00:00"
export const distance = formatDateDistance(date1, date2)

## Both ISO and date Object

formatDateDistance allow use of both ISO 8601 format and date object, and can also compare them

```javascript
    const date = new Date(1524614400000)
    export const date2 = "2020-03-19T00:00:00"
    export const distance = formatDateDistance(date, "2018-05-10T00:00:00Z")
```
