# formatMoney
formatMoney is a bento tool that helps you format numbers to currency. It takes a value, and options.

### Import
All exports under formatting is imported via formatting. to use a spesific formatter you would
use ```formatting.FormatterYouWant```


```javascript
import { formatting } from "@staccx/bento"
```
usage
```javascript
formatting.formatMoney
```

## Standard formatMoney
This is the standard formatMoney with no options, as standard, formatMoney adds a space as thousand separator

<Preview>
  <Story name="Standard">
   <CurrencyText>{formatMoney(1000)}</CurrencyText>
  </Story>
</Preview>

## Norwegian
Here is an example of how we generally would format a norwegian currency string. Here we use locale from
options, to format the currency to the norwegian standard

<Preview>
  <Story name="Norweigan format">
   <CurrencyText>
   {formatMoney(100000, { locale: "nb-NO", suffix: ",-"})}
   </CurrencyText>
  </Story>
</Preview>

<Preview>
  <Story name="kr suffix">
   <CurrencyText>
   {formatMoney(100000, { locale:"nb-NO", suffix: "kr", style: STYLE.decimal})}
   </CurrencyText>
  </Story>
</Preview>
