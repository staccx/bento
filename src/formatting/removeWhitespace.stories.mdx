# RemoveWhitespace
RemoveWhitespace removes all white space in the string that is passed to the function

### Import
All exports under formatting is imported via formatting. to use a spesific formatter you would
use ```formatting.FormatterYouWant```


```javascript
import { formatting } from "@staccx/bento"
```
usage
```javascript
formatting.removeWhitespace
```

## Standard removeWhitespace
Remove leading white space

<Preview>
  <Story name="Standard">
   <p>{removeWhitespace("        Test")}</p>
  </Story>
</Preview>

## Remove trailing
Remove trailing white space

<Preview>
  <Story name="remove trailing">
   <p>{removeWhitespace("Test         ")}</p>
  </Story>
</Preview>

## Remove all white space in string
"This will be hard to read, but that is intended, because i have removed all white space" becomes:
