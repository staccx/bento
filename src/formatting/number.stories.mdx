# Format number
Number formatter for all your needs

### Import
All exports under formatting is imported via formatting. to use a spesific formatter you would
use ```formatting.FormatterYouWant```


```javascript
import { formatting } from "@staccx/bento"
```
usage
```javascript
formatting.formatNumber
```

### Defaults
```javascript
const defaults = {
  style: STYLE.decimal,
  currencyDisplay: CURRENCY_DISPLAY.symbol,
  suffix: null,
  unitDisplay: UNIT_DISPLAY.short,
  currency: "NOK", // http://www.currency-iso.org/en/home/tables/table-a1.html,
  useGrouping: true,
  locale: "nb-no",
  unit: null,
  minimumFractionDigits: 0
}
```
### Usage of number constants
```javascript
  formatting.CURRENCY_DISPLAY,
  formatting.STYLE,
  formatting.UNIT_DISPLAY,
  formatting.UNITS
```
