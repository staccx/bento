# Truncate
Turncate helps deside the length of a string

### Import
All exports under formatting is imported via formatting. to use a spesific formatter you would
use ```formatting.FormatterYouWant```


```javascript
import { formatting } from "@staccx/bento"
```
usage
```javascript
formatting.turncate
```


```javascript
const longString = "The quick brown fox jumps over the lazy dog"
const shortString = "The quick brown fox"
```

## Standard truncate

export const longString = "The quick brown fox jumps over the lazy dog"
export const shortString = "The quick brown fox"

<Preview>
