# Remove Whitespace

<Subtitle>
  This document describes the use of the `removeWhitespace` function
</Subtitle>

## Requirements

`@staccx/bento >= 14.0.0`

## Usage

```js
import { removeWhitespace } from "@staccx/bento"
const value = removeWhitespace("   test")
```
