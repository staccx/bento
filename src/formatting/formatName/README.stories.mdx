# Format Name

<Subtitle>
  This document describes the use of the `formatName` function
</Subtitle>

## Requirements

`@staccx/bento >= 14.0.0`

## Usage

```js
import { formatName } from "@staccx/bento"
const formattedName = formatName("john doe")
```
