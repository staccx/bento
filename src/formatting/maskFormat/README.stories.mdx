# Mask Format

<Subtitle>
  This document describes the use of the `maskFormat` function
</Subtitle>

## Requirements

`@staccx/bento >= 14.0.0`

## Usage

```js
import { maskFormat } from "@staccx/bento"
const maskedValue = maskFormat(92284082, "XX-XX-XX-XX")
```

## Arguments

| Argument name                                     | Type                         | Default value | Description |
| ------------------------------------------------- | ---------------------------- | ------------- | ----------- |
| `value` <Badge status="positive">required</Badge> | `string or number or bigint` |               |             |
| `mask`                                            | `string`                     |               |             |
